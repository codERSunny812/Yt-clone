# Installation
1. Clone the repository:

 ```bash
  git clone "https://github.com/codERSunny812/Yt-clone.git"
```
2. go to the projectory

```bash
 cd youtubeClone
```

3. Install the dependencies

```bash
 yarn install
```
4. go to the projectory

```bash
 yarn dev
```
